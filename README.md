#  My To-Do List App

A simple and clean To-Do List web app built using React.  
This project was created as part of my learning journey to understand core React concepts and build hands-on experience.

---

##  Features

-  Add new tasks  
-  Delete (mark as done) tasks  
-  Instant UI updates using React state  
-  Simple and beginner-friendly logic  

---

## Tech Stack

- React (useState, components, props)
- JavaScript (ES6)
- HTML & CSS
- Bootstrap (for basic styling)

---

##  Purpose of this Project

This project was built for **practice and learning**, with the help of a tutorial, to:
- Understand how React components work  
- Learn state management using `useState`  
- Practice passing data using props  
- Get comfortable building small functional apps  

---

##  Deployment

This app is deployed using Vercel.  
https://to-do-list-49vd.vercel.app/

---

##  What I Learned

- How to create and structure React components  
- Handling form inputs using controlled components  
- Managing and updating state dynamically  
- Debugging common React errors

This is a beginner-level project built for practice.  
The focus was on learning and understanding concepts rather than building a production-level app.
